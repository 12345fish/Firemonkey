Firemonkey Plugin Demo
http://www.fmxexpress.com/

This supports two kinds of plugins. A URL plugin and a Pascal script plugin.

In the URL plugin you can add a URL which will be launched when you run the plugin. 
The URL launches in the external browser but could also be launched in an in app browser.

In then Pascal script plugin you can add a Pascal script file which will be launched then you run the plugin. 
The Pascal script is interpreted by paxCompiler and can access the main form. The sample script will also open it's own form.
