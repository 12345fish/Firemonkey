Title: IBLite

Brief Description: This code snippet shows you how to populate a listview from an IBLite database using LiveBindings.


To deploy this snippet you need to use the IBLite license, following the instructions that were emailed along with the 
RAD Studio license (or trial license). For more information on deploying the license file along with the application, you can 
read

http://docwiki.embarcadero.com/RADStudio/XE6/en/IBLite_and_IBToGo_Test_Deployment_Licensing 
for additional information.
